# GwtRestPerf

## Goal

Compare **performance** of [RestyGWT][1] and pure [GWT][2] for rest calls.

### Topics

1. Serialization/deserialisation speed
2. Number of simultaneous calls supported
3. Support for Jax-rs annotations
